# Bank Loan Default Risk Analyzer

## Overview

The Bank Loan Default Risk Analyzer is an end-to-end Data Analytics project designed to analyze borrower risk patterns and identify high-risk loan applicants using SQL and Excel. The project combines data cleaning, risk scoring, KPI analysis, and interactive dashboarding to provide business-driven insights for loan approval and risk management.

## The Project Includes:
- SQL-based risk analysis
- Interactive Excel dashboard
- Dynamic Loan Applicant Risk Checker
- Business insights for decision-making

## Tools & Technologies Used:
- MySQL – Data Cleaning & Risk Analysis
- Excel – Dashboarding & Interactive Reporting
- Pivot Tables & Charts – KPI Visualization
- Risk Scoring Model – Rule-based applicant evaluation

## Dataset Information

The dataset contains loan applicant information such as:
- Applicant Age
- Annual Income
- Loan Amount
- Credit Score
- Employment Type
- Loan Purpose
- Default Status

## Key Features

### SQL Analysis
Performed advanced SQL analysis including:
- Overall Default Rate
- Default Rate by Credit Band
- Loan Purpose Risk Analysis
- Employment-wise Risk Analysis
- Applicant Segmentation
- Risk Ranking using Window Functions
- Loan-to-Income Ratio Calculation
- Rule-Based Risk Scoring Model

### Risk Scoring Model
A custom risk scoring model was developed using:
- Credit Score (40%)
- Loan-to-Income Ratio (35%)
- Age Risk (15%)
- Employment Stability (10%)

#### Applicants are categorized into:
-  High Risk
-  Medium Risk
-  Low Risk

### Interactive Dashboard
The Excel dashboard provides:
- KPI Cards
- Default Rate Analysis
- Credit Band Risk Analysis
- Loan Purpose Insights
- Risk Distribution Visualization
- Employment-based Risk Analysis
- Interactive Slicers for Filtering

### Loan Applicant Risk Checker
A separate interactive tool was created where users can:
- Enter applicant details
- Automatically calculate:
  - Loan-to-Income Ratio
  - Credit Band
  - Age Group
  - Risk Score
  - Risk Label

## Key Business Insights
- Younger applicants exhibit higher default rates.
- Business loans show elevated repayment risk.
- Low credit-score borrowers dominate high-risk segments.
- Unemployed applicants display the highest average risk scores.
- Auto and Education loans demonstrate moderate default behavior.
